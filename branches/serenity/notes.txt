Git notes:

To check out a entire subversion repos:
$ git svn init -t tags -b branches -T trunk <url>
$ git svn fetch

Making local commits
$ git commit -a

Creating a new branch to a new directory
$ git clone -l <working dir> <new branch dir name>

Creating a new branch in the current working dir
$ git checkout -b <branchname> rmotes/trunk

Updating the working dir (equiv: svn update)
$ git svn rebase
[or to save local changes first]
$ git-commit -a -m "Temporary commit"
$ git svn rebase
$ git reset --mixed HEAD^

Commit back to svn repo
$ git-svn dcommit
